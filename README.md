# Sales Analytics Dashboard (Power BI)

## 📊 Project Overview
This dashboard provides a comprehensive analysis of sales performance, trends, and customer insights to support business decision-making.

## 🔑 Key KPIs
- Sales YTD: 5.21M
- Cost of Goods Sold (COGS): 3.52M
- Gross Profit: 1.69M
- Quantity Sold: 0.95K

## 📈 Key Insights
- Strong sales performance observed in February and May
- Furniture and Home Appliances are top-performing categories
- Zulfi and Merut stores contribute highest revenue
- Customer analysis highlights top revenue-generating clients

## 📊 Dashboard Features
- Sales vs Target Analysis
- Monthly, Weekly & Yearly Trends
- Category & Store Performance
- Customer & Product Analysis
- Growth Analysis using advanced visuals

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling
- Business Intelligence
